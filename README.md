# Nail IB Frontend

A premium, high-performance landing page for IB students. Built with a focus on "Executive Academic" aesthetics and smooth interactive experiences.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **Icons**: Phosphor React

## Key Features
- **Bento Grid**: Data-driven cards with interactive topics and exemplars.
- **Educator Showcase**: Infinite carousel featuring top IB specialists.
- **Adaptive UI**: Dark-mode enforced design with high-contrast accessibility boosts.
